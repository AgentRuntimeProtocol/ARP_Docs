---
title: Author Atomic Nodes
---

Atomic nodes are capabilities that “just execute”: given inputs, run bounded work and return outputs.

This section will grow around the server SDK + templates for building atomic nodes with:
- strict request/response envelopes,
- idempotency and side-effect classification,
- durable artifacts and predictable error models.
